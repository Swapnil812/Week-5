# Week-5
In this assignment, I learned the fundamentals of Apache Spark and worked with Spark DataFrames to perform end-to-end data processing. I understood the limitations of traditional MapReduce and how Spark's in-memory computing makes big data processing much faster and more efficient. Using PySpark, I loaded a dataset, cleaned it by removing duplicate records and handling null values, applied various filtering conditions, and performed aggregations such as count, sum, average, minimum, and maximum. I used groupBy() to analyze data by different categories, modified the schema by renaming and casting columns, and learned about DataFrame immutability and wide transformations like shuffle. Finally, I combined all these operations into a complete data processing pipeline that transformed raw data into meaningful insights, demonstrating the typical workflow used in real-world data engineering projects.
